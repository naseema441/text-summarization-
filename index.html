Text Summarization Tool

This tool uses extractive and abstractive techniques to summarize large blocks of text.

It’s powered by Python, Gradio, Sumy (LSA), and Facebook’s BART model.

To run it:

git clone https://github.com/your-username/your-repo.git
cd your-repo
pip install -r requirements.txt
python your_script.py
Try it Live (if deployed)
